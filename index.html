ISR Value Calculator Gem Builder

Follow these steps to create your own instance of the Value Calculator Gem.

To begin, you'll need to access your Gemini instance and create a new Gem. This is where you'll "teach" the AI its role as the Value Calculator.

  1. Access Gemini: Navigate to your organization's Gemini platform.
  2. Start New Gem: Look for an option like "Create New Gem" or "Build a Custom AI" within the Gemini interface. This is typically found on the main dashboard or a dedicated "Gems" section.
  3. Name Your Gem: Give your Gem a descriptive name, such as "Simpro Value Calculator" or "ISR Value Gem."
  4. Proceed to Configuration: Once named, you'll be prompted to configure the Gem's instructions and knowledge base, which are covered in the following steps.

The effectiveness of the Gem depends entirely on the information it has access to. Collect all relevant Simpro documents. The original Gem was created using the following knowledge base:

Using The Value Calculator

Learn how to query the Gem and understand its output for effective value conversations.

Input: How to Query the Gem

To get the most effective value calculation, the ISR should provide the following information to the Gem, formatted with a bullet point for each item. You can copy and paste this structure directly into the Gem.

  • "Please provide a value calculation for a **[Customer Type]** prospect."
  • "They are a **[Industry]** company called **[Company Website]**."
  • "They have **[Number]** field technicians."
  • "Their main pain points are **[Pain Point 1]**, **[Pain Point 2]**, and **[Pain Point 3]**."
  • "The estimated annual cost of their current inefficiencies is **[Amount]**."

Example Filled-In Prompt:

  • "Please provide a value calculation for a **Residential** prospect."
  • "They are an **HVAC** company called **Polar Air**."
  • "They have **50** field technicians."
  • "Their main pain points are **inefficient scheduling**, **manual invoicing**, and a **lack of job costing visibility**."
  • "The estimated annual cost of their current inefficiencies is **$300,000**."

Output: Understanding the Value Breakdown

When you query the Gem with the structured input, it will provide a detailed value calculation. This output is designed to be clear, concise, and directly applicable to your sales conversations.

  • Executive Summary: A one-paragraph summary of the key findings.
  • Key Value Metrics: A detailed breakdown of quantifiable results (e.g., "Up to 50% reduction in admin time," "$100k+ recovered margin").
  • Supporting Case Cases: A reference to one or two case studies from the knowledge base that support the metrics.
  • Talk Track Recommendation: A suggestion on how to frame this information, referencing the enablement guidelines in Part 2.
  • Monthly Value Generated: A section detailing the potential monetary value of hours and new tasks accomplished.
  • Monthly Cost of Simpro Licenses: A section outlining the cost of the required licenses.
  • Net Monthly Impact: The final calculated net gain or loss for the client.

Wow Your Prospect with a Visual Value Breakdown!

After the Gem provides the value breakdown, you can use Gemini Canvas to quickly transform this data into a compelling infographic to share with your prospect. This visual summary can significantly enhance your presentation and leave a lasting impact.

  1. Copy Gem Output: Copy the entire value breakdown output from the Gem.
  2. Go to Gemini Canvas: Open a new Canvas in your Gemini instance.
  3. Paste & Prompt: Paste the Gem's output into the Canvas. Then, prompt Gemini to "Create an infographic summarizing this value breakdown for [Prospect Company Name], highlighting the net monthly impact and key savings areas."
  4. Review & Share: Review the generated infographic. You can make minor edits directly in Canvas. Once satisfied, share it directly with your prospect or use it in a follow-up email.
